### 权限控制
```text
菜单访问权限和操作权限放在一起，但是二者在使用时分割开；
可以存在有菜单权限，但是没有操作权限；
也可以有此菜单的操作权限，但是没有此菜单页面的访问权限
考虑到以下情况：
    1、可能有业务需求，某个角色可以从一个页面查看另一个页面的内容，并且有部分操作权限
        例如：下属查看领导的审批结果，并且拥有同意审批和申诉审批两个功能；
             这两个功能是领导不具备的
```
